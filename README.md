# MABe-Challenge-2025 ML Group Project

## Dataset Preparation

Extract the kaggle datasets into the `data/` folder at the project root:

After extraction, your project directory should look like this:

```
MABe-Challenge-2025/
├── data/
│   ├── train.csv                    # Metadata for training videos
│   ├── test.csv                     # Metadata for test videos
│   ├── sample_submission.csv        # Submission format template
│   ├── train_tracking/              # Pose tracking data (training)
│   ├── train_annotation/            # Behavior labels (training)
│   └── test_tracking/               # Pose tracking data (test)
│
├── mabe_mouse_behavior_download.ipynb
└── ......
```
